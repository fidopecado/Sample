# Learning Tracker
 
A simple web app to track daily learning logs, ticket activities, problems encountered, problems solved, and portfolio progress.
 
## Features
- Add learning logs
- Required form validation
- Display recent logs as cards
- Newest logs appear first
- Empty state when there are no logs
- Summary counters for total, solved, and submitted logs
 
## Tech Stack
- HTML
- CSS
- JavaScript
 
## Current Version
Version 1 focuses on basic frontend structure and DOM manipulation.
